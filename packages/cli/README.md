## shell简化git命令

#### Installing

```sh

npm i @f-git/cli -g

```

#### Use


###### pull

> 

```sh

ff pull
# 或
ff pl

```

###### push

> 

```sh

ff push
# 或
ff ph

```


###### checkout

> 

```sh

# ff checkout <branch>

ff checkout dev
# 或
ff ck dev

```


###### commit

> 

```sh

ff commit
# 或
ff cm

```


###### stash

> 

```sh

ff stash

ff stash pop

# 或

ff s

ff s pop

```
