## shell简化git命令

#### Installing

```sh

npm i fcg -g

```

#### Use


###### pull

> 

```sh

ff pull

```

###### push

> 

```sh

ff push

```


###### checkout

> 

```sh

# ff checkout <branch>

ff checkout dev

```


###### commit

> 

```sh

ff commit

```


###### stash

> 

```sh

ff stash

ff stash pop

```
